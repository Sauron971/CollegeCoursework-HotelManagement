# BaronessaHotel-XAF-CSharp

Учебный проект системы управления гостиницей, выполненный в рамках курсовой работы в колледже.

## О проекте

Проект разработан на языке **C#** с использованием фреймворка **DevExpress XAF (eXpressApp Framework)**.

Цель работы - создание модели базы данных для гостиничной системы и получение базового функционала управления данными (CRUD) средствами XAF.

Проект не является готовым приложением для запуска или коммерческого использования. Это исходный код учебной работы, представленный в виде проекта Visual Studio.

## Используемые технологии

* **C#**
* **.NET**
* **DevExpress XAF**
* **Entity Framework / ORM**
* **Visual Studio**
* **SQL Server (через настройку XAF)**

## Реализовано

В проекте были созданы модели данных для предметной области гостиницы:

* сущности базы данных;
* связи между объектами;
* структура хранения данных.

После создания моделей фреймворк XAF автоматически сформировал:

* интерфейс управления данными;
* операции создания, просмотра, изменения и удаления записей (CRUD);
* взаимодействие с базой данных.

## Особенности

Данный проект отличается от стандартного подхода, который использовался в учебной группе.

Вместо разработки интерфейса вручную на Windows Forms был выбран фреймворк XAF.

Использование XAF позволило сосредоточиться на проектировании моделей данных и значительно сократить количество ручного кода.

## Запуск проекта

На данный момент проект не содержит готовой сборки или опубликованного приложения.

Для работы требуется:

1. Установить Visual Studio.
2. Установить необходимые компоненты .NET и DevExpress XAF.
3. Открыть solution-файл проекта.
4. Настроить подключение к базе данных.
5. Запустить проект через Visual Studio.

## Статус проекта

Проект завершён в рамках курсовой работы.

Дальнейшая разработка и расширение функционала не планируются.
